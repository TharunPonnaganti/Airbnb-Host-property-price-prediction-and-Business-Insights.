# Airbnb Pricing Analytics

## Overview
This project analyzes Airbnb listings in New York City to understand how listing characteristics such as neighborhood, room type, bedrooms, and other host- and property-level features influence price.

The project frames pricing as a practical data science problem: using structured listing data to estimate listing prices and support more informed pricing decisions.

## Business Problem
For Airbnb hosts, pricing is a balance between competitiveness and profitability. Underpricing can leave revenue on the table, while overpricing can reduce bookings and occupancy.

This project approaches the problem from both an analytical and business perspective by:
- identifying the major drivers of listing price
- preparing listing data for predictive modeling
- evaluating how data-driven pricing can support host decision-making

## Objective
- Understand which features are most associated with listing price
- Build a repeatable workflow for price prediction
- Generate insights that could support pricing recommendations

## Dataset
The analysis uses Airbnb listing data for New York City, including variables such as:
- neighborhood / location
- room or property type
- bedroom-related attributes
- host and listing characteristics

## Analytical Approach
1. Exploratory data analysis to understand price distribution and feature relationships
2. Data cleaning and preprocessing
3. Feature engineering and transformation
4. Predictive modeling for price estimation
5. Interpretation of results for business use

## Potential Use Cases
- Host pricing support
- Revenue optimization analysis
- Competitive market benchmarking
- Internal pricing recommendation workflows
