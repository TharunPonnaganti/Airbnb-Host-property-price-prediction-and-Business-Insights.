# Airbnb Pricing Analytics

## Overview
This project analyzes Airbnb listings in New York City to understand how listing characteristics influence price and to build a predictive pricing workflow.

## Business Problem
Airbnb hosts often struggle to price listings optimally. Underpricing reduces revenue, while overpricing reduces occupancy.

## Objective
- identify key drivers of listing price
- build a predictive model for price estimation
- support data-driven pricing decisions

## Approach
1. Exploratory data analysis
2. Data preprocessing
3. Feature engineering
4. Predictive modeling
5. Insight generation

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Key Outcome
Built a pricing model that captures relationships between listing features and price, enabling structured pricing analysis.

## Why This Matters
Demonstrates real-world pricing analytics and ML workflows for business decision support.
